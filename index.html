Informações de Deploy

Esta verificação está configurada para trabalhar com:

Status 1: Telefone válido e associado a uma conta

Status 2: Telefone não associado a uma conta

Plataforma de deploy: Vercel

Para implementar a verificação real, você precisará:

  • Configurar uma API serverless na Vercel
  • Implementar a lógica de verificação real
  • Conectar com a API do Kako
